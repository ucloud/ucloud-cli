ucloud project create
---------------------

Create project

Synopsis
~~~~~~~~


Create project

::

  ucloud project create [flags]

Examples
~~~~~~~~

::

  ucloud project create --name xxx

Options
~~~~~~~

::

  --help, -h               help for create 

  --name     string        The name of project 

  --parent-id     string   The parent project id
